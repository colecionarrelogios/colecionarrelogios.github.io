# Colecionar Relógios — site estático

Estrutura pronta para GitHub Pages.

## Como publicar
1. Extraia o conteúdo do ZIP.
2. Envie todos os arquivos para a raiz do seu repositório no GitHub.
3. Vá em **Settings > Pages**.
4. Em **Source**, selecione **Deploy from a branch**.
5. Escolha a branch principal e a pasta **/(root)**.
6. Salve.

## Ajustes recomendados
- Trocar `https://seuusuario.github.io/colecionar-relogios/` pela URL final.
- Inserir link real do WhatsApp em `contato.html`.
- Substituir os textos de coleção pelos seus relógios reais.
- Adicionar novos artigos no blog.
